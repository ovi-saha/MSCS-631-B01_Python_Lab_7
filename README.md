# MSCS-631-B01_Python_Lab_7
This repo is Python Lab &amp; for Advanced Computer Networks (MSCS-631-B01) - Second Bi-term
## How To Run
- To run this project, keep all the files in the same directory.
- Then run the command **python3 Server.py 5544** in a terminal to open the server.
- Then open a new terminal and run **python3 ClientLauncher.py 127.0.0.1 5544 25000 movie.Mjpeg** to run the client.
- 5544 is the port number. You can use any open port.
